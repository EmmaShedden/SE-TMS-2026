__Summary:__
We include all replication materials, including survey materials, codebook, qualitative survey analysis, quantitative survey analysis, performance data management scripts, and performance data analysis scripts.

- **_data-analysis_** - Contains survey and performance analysis scripts/documentation 
- **_stimuli_** - Contains raw stimuli form. Stimuli in its presentation form can be found in the analysis scripts
- **_surveys_** - Contains all surveys (pre-screening, demographics, and post-study survey). The post-study survey is the one analyzed qualitatively in our paper


For _data-analysis_ folders (specifically in the Performance Analysis and Survey Analysis Folder), we have more verbose README documents to help guide the interested reader in viewing or reproducing the results.
