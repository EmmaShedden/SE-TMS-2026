# Transcranial Magnetic Stimulation for Software Engineering: Causality, Efficiency and Accuracy around Neurostimulation for Programming

Statistical analysis replication package for this paper submission. Arm 1 (spatial reasoning; `arm_1/`) is mostly identical to our [earlier conference publication](https://github.com/hammad-a/ICSE24_TMS). Arm 2 (working memory; `arm_2/`) is entirely new material.

This repository contains experiment stimuli, participant surveys, anonymized datasets, and data analysis scripts.

Preprocessed data is provided for both arms, so you can run our statistical analysis scripts. However, raw data may be partially or completely redacted to protect human subjects' anonymity. For access to redacted raw data, please contact the authors.
